HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • inline CSS: to style an element directly in the HTML file, useful for quick and permanent changes
  • margin property: indicates how much space we want around the outside of an element.
  • padding property: adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger.
  • class attribute as the selector: to add styles to certain elements, but not all the elements.

Git

  • git status: checks what branch we are currently on.
  • git checkout: -b branch-name: creates a new branch and switches to it.
  • git config --global user.name: configures user name across all repositories.
  • git config --global user.email: configures user email across all repositories.
  • git commit -m: commit your staged content as a new commit snapshot.

JavaScript

  • JavaScript is a powerful programming language that can add interactivity to a website.
  • Browser Application Programming Interfaces (APIs): built into web browsers, providing functionality such as dynamically creating HTML and setting CSS styles; collecting and manipulating a video stream from a user's webcam, or generating 3D graphics and audio samples.
  • Control flow: the order in which a computer executes code in a script.
  • Array: learn how to store a group of related data.
  • Commenting out code: makes it inactive without deleting it, just in case you need it later.
